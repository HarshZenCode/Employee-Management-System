# Employee Management System

## 📌 Project Overview

Employee Management System is a Java-based application that helps manage employee records efficiently. It allows users to add, update, delete, and search employee details using a MySQL database.

---

## 🚀 Features

- ➕ Add Employee
- ✏️ Update Employee
- ❌ Delete Employee
- 🔍 Search Employee by ID
- 📋 View All Employees
- 💾 MySQL Database Integration
- ✅ Input Validation
- ⚡ JDBC Connectivity

---

- Email
- Phone Number
- Salary

---

## ▶️ How to Run

1. Clone the repository

```
git clone https://github.com/HarshZenCode/Employee-Management-System.git
```

2. Open the project in IntelliJ IDEA.

3. Create the MySQL database.

4. Update the database username and password in the JDBC connection file.

5. Run the project.
